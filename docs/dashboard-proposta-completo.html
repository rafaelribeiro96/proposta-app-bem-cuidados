build Mitigação de Desafios Técnicos

Botão de Emergência (Pânico) expand_more
No WebApp, contornamos a limitação de push background enviando gatilhos via WhatsApp Business API ou SMS. O sistema dispara alertas automáticos para a central e família, garantindo que o pedido de ajuda chegue mesmo que o app não esteja aberto na frente do usuário.
Uso da Câmera e GPS expand_more
PWAs modernos têm acesso pleno à câmera e geolocalização precisa através do navegador. O cuidador pode bater fotos do paciente (curativos, alimentação) e o sistema registra a coordenada exata no momento do registro clínico.

Plano de Evolução: Do WebApp ao Nativo

Mês 1-3: Validação & Caixa

Operamos com o WebApp para validar se o modelo de check-in funciona. Corrigimos fluxos com base no uso real dos cuidadores sem custos extras.

Mês 4+: Consolidação

Com o sistema estável, "encapsulamos" o código em um app nativo para as Stores. O usuário não sente a mudança, os dados são os mesmos, mas a marca ganha autoridade.

check_circle

Utilizando o Supabase como backend, garantimos que qualquer mudança no front-end (de Web para Mobile) seja transparente para o banco de dados. Zero perda de informação.
